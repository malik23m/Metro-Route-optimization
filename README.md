#  Metro Route Optimization

A Data Structures and Algorithms (DSA) project in Java that finds the shortest path between metro stations using graph algorithms.

---

##  Features

-  Find shortest route between two metro stations
-  Calculate minimum travel time
-  View all metro stations in the network
-  Interactive menu-driven interface
-  Dijkstra's algorithm implementation
-  Graph data structure for metro network

---

##  Algorithms Used

- **Dijkstra's Algorithm** - For finding the shortest path
- **Graph Data Structure** - For representing metro network
- **Priority Queue** - For efficient path finding

---

##  Technologies Used

- **Java** - Core programming language
- **OOP Concepts** - Classes, Objects, Inheritance
- **Collections Framework** - ArrayList, PriorityQueue
- **File I/O** - For metro data storage
